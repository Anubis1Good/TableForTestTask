База данных использовалась PostgreSQL

Для проверки необходимо создать БД с именем "table_for_test_task1"
или изменить базу данных в файле db.js

Затем выполнить скрипт database.sql в PostgreSQL

Выполнить команду 'npm install'
Выполнить команду 'npm run client:install'

Все приложение целиком запускается командой "npm run dev"
